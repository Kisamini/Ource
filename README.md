# Ource : Our voice social media service

http://52.79.235.41:8080/





### What is the Ource?

Ource(Our voice project)는 글, 미디어 파일만 공유하던 기존의 소셜 미디  어 서비스와는 달 리, TTS (text to speech) API를 통한 유저의 게시물 읽기  기능 최대의 아이 덴티티로 확립하여 개발한 소셜 미디어 서비스입니다.
